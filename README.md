## Local Development
```shell script
# 用意
docker compose build

# 起動
docker compose up -d

Laravelプロジェクトに暗号化キー　設定
php artisan key:generate

# シャットダウン
docker-compose down

# 開始
docker-compose start

# 停止
docker-compose stop

# 再起動
docker-compose restart
```
